TOFSTUFF is a collection of irregular stuff (audio-art instruments) created by Thomas O Fredericks in Verdun, Quebec, Canada.
Some of the stuff generates sounds, some catalyze destruction, others are simple utilities.
While none of the stuff can be bought, most are open-sourced and can be DIYed.
If you live in Montreal, contact me to see if you could pickup a kit!

le SQUARE is a simple MIDI thru box that distributes 1x 3.5mm MIDI input to 4x 3.5mm outputs. It is powered through USB and properly opto-isolated. It is rather small as a 2" x 2" square.

Comes with 4 designs for the faceplate. Select the one you like the most.

Showdown is designed to be the FINAL(showdown) module in your Eurorack modular chain. It brings Eurorack levels DOWN to Line levels.

Showdown is a 2HP Eurorack module that combines two Eurorack level inputs to a single stereo line level output. A 30mm travel slider controls the stereo line level ouput volume.


Puppeteer is a multi function Eurorack 8HP module. It can convert MIDI to Eurorack and provides an additonnal 4 CV outputs for envelopes, presets, random, etc.

The panel can be flipped to the male or female stereotypes.


Supernova is a 2HP Eurorack Low Pass Gate (LPG), with extreme settings for resonance and distortion. It is analog and provides different combinations of capacitors that create different low pass filter slopes. It does not use a vactrol, as is typical, but an actual LED (that shines through the panel) and light-sensitive resistors. The design is very reactive to gate times and amplitude variation and offers natural rise and decay slopes. The toggles allow the player to reconfigure or break the circuit for total super explosive distortion and resonance.

MozArd is a digital micro-synthesizer that is easy to assemble in less than 2 hours. It features a 12-key keyboard which allows you to play all the notes of an octave and 3 potentiometers to adjust the sounds produced by the micro-synthesizer. For advanced users, these sounds can be reprogrammed.

MozArd is based on Mozzi and an Arduino Nano microcontroller.

You can build your own from the Git Source Repository

The Op-Synth is an open source micro (3" wide by 3" long) modular analog synthesizer kit designed to facilitate the understanding of electronics, audio synthesis, audio theory, and filtering.

Graphical design and illustrations by Denis Raby

You can build your own from the Git Source Repository
